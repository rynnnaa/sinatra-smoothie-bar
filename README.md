#Sinatra Smoothie Bar is a Sinatra MVC application that allows users to create and edit Smoothie recipe ideas.

## Installation
Fork and Clone Repository.

Run following commands in terminal

```zsh
bundle install
```

```zsh
rake db:migrate

```zsh
shotgun
